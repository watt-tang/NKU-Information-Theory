
# 信息论

**授课时间：** 2026 Spring

**授课老师：** 宁博

## 总体评价

整体难度很高，应该算是密码同学本学期比较硬核的一门课。宁博老师的上课风格主要是板书推导，课程里有大量数学推导，也会用到不少概率论与数理统计相关的知识，整体风格和宁老师另一门《数理逻辑与图论》比较像。

这门课非常不适合期末突击。因为考试不是那种背几个定义、刷几道题就能混过去的类型，而是基本全是数学推导题。很多题目题干就一小行，然后给你一大页空白让你推。如果平时没怎么听、没理解推导过程，期末想硬背下来其实挺难的，考场上很容易直接无从下手。就算考试时每道题都直接有思路，最后也剩5min 时候才完全写完。

参考书可以看 Thomas Cover 的黑皮书《信息论基础》，上面有一些定理证明和课后习题，不过个人感觉课上板书还是很重要的。

## 平时情况

本学期平时没有固定作业，所以不像一些课那样可以靠作业题反推考试重点。平时可以自己整理整理课堂笔记，据说整理得比较好的笔记发给老师，平时分可能会比较高，但具体平时分构成我也不是很确定，这一点存疑。

签到方面，整个学期大概签到了两三次。老师有时候会在人比较少的时候突然签到，而且是前后门锁门那种，所以这门课不太建议长期缺勤，不然容易突然平时分暴毙。

## 学习建议

最重要的建议就是：**平时尽量跟着老师把推导听懂，不要全指望期末突击。**

信息论很多证明本身其实不一定特别长，但是它很依赖你对概念之间关系的理解，比如熵、条件熵、联合熵、互信息、相对熵、典型集、信道容量这些东西。如果这些概念之间没串起来，期末看到题目就会感觉每个字都认识，但是不知道从哪开始写。

本学期倒数第二次课，老师会强调“应知应会的内容”。这部分非常重要，基本覆盖了期末考试中大概 80 分左右的内容，所以这节课一定要认真听、认真记。虽然这门课整体难度确实很高，但根据我自己和身边同学的情况来看，只要平时认真学了，期末把老师强调的重点内容过到位，最后给分还算不错（至少相对于上学期图论

## 考试内容

本学期期末考试大概是六四开，下面题目顺序不一定完全准确。

### 1. 熵、互信息与相对熵相关证明

第一问要求写出 $n$ 个随机变量联合熵的链式法则。

第二问证明熵的最大值原理。

第三问证明互信息非负性。

后两问本质上都是 Jensen 不等式或者相对熵非负性的应用。其中互信息非负性可以用相对熵来证明，属于比较核心的基础证明。

### 2. 三元互信息正负性

要求用韦恩图或者其他方式说明三元互信息可能为负数，并且要给出一个具体例子。

这个例子老师上课好像没有直接讲过，需要自己构造。如果考前没准备过，考场上临时想会比较难，所以建议提前背一个能说明三元互信息为负的标准例子。

### 3. 利用韦恩图证明熵函数等式

题目要求说明一个等式。老师上课说过不需要严谨推导，用三个随机变量的韦恩图就行，这个整体比较简单，只要熟悉联合熵、条件熵、互信息在韦恩图中的对应区域，基本就能写出来。

这类题主要是别把区域看反，画清楚之后就比较白给，上课时候老师还让同学上过黑板，大家应该有印象。

### 4. 熵的公理化系统

要求利用组合原理推导熵的公理化系统。

这个题相对偏证明，需要掌握老师课上从组合计数角度引出熵函数形式的推导过程。不是特别适合临场发挥，最好考前整理一版完整证明。

### 5. 二元 Huffman 编码

给定概率，要求构造二元 Huffman 编码树，并计算平均码长。

这题属于比较常规的计算题，难度不算大，但是树的合并过程要画清楚，最后平均码长不要算错。Huffman 这种题属于能拿就一定要拿的分。

### 6. 典型集

要求写出典型集的定义，并证明典型集的五条性质。

本次重点考察了第二条、第三条和第五条性质。典型集这一部分非常重要，属于老师强调过的核心内容。

### 7. McMillan 定理证明

好像考查的是 McMillan 定理的证明。

这个题需要熟悉唯一可译码、码长和 Kraft 型不等式之间的关系。

### 8. 推广打字机信道与信息处理不等式

考查了推广打字机信道容量为 $\log(n/2)$ 的推导，同时还涉及利用信息预处理不等式的证明。

个人感觉这一题里面有些步骤和第九题的证明思路有一定重复。推广打字机信道这块不要只背结论，最好理解为什么容量会是 $\log(n/2)$。

### 9. 信道编码定理逆定理

考查信道编码定理逆定理的证明，同时要求说明什么是离散无记忆信道，以及如何理解“无记忆性”和“无反馈性”。

这题属于全卷中比较核心的大题，需要对信道编码定理的整体逻辑有比较清晰的理解。

## 总结

总体来看，2026 Spring 的信息论考试非常重视证明和推导，不太考机械记忆型题目。老师最后课上强调的重点内容非常关键，基本决定了期末能不能稳住大部分分数。

复习时建议优先掌握下面几块：

* 熵函数基本性质，公理系统
* 互信息与相对熵
* Huffman 编码
* Kraft / McMillan 定理
* 典型集
* 信道容量
* 信道编码定理及其逆定理
